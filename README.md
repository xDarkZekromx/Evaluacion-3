# Evaluacion1-AgendaDeContactosPersonal
Evaluacion numero 3

▪ Descripción del caso. 
 
 Se nos ha solicitado la creación de una agenda de contactos para un cliente,
 dicho cliente, nos ha solicitado que tenga una estetica similar a balatro,
 por lo que como equipo de desarrollo, llevamos a cabo esa mision a traves de angular.

▪ Instrucciones de instalación y ejecución local. 

 Para ejecutar este archivo, lo levantamos a traves de un servicio gratuito en render,
 render se encarga de gestionar la pagina para comunicarse con SupaBase, siendo esta última
 la que se encarga de manejar la base de datos y todo el flujo que conlleve.

 link de este archivo levantado en render: https://evaluacion-3.onrender.com/

 Este archivo puede ser levantado igualmente a traves de comandos locales,
 como lo puede ser "python manage.py run server"

▪ Rutas principales de la API. 

 Al ingresar a la página veremos algo simple, sin mayor detalle, pero nos sirve para ver que la
 página se encuentra en funcionamiento, pero esconde algunos links ocultos, como lo puede ser:
 /contactos
 /personas
 /api

▪ Forma de autenticarse (endpoints, ejemplos de payload). 

 Para acceder a la API (/api), tenemos como credenciales "admin" y "Contraseña", sin esas credenciales
 no podremos acceder a las distintas vistas con las que cuenta la pagina.
